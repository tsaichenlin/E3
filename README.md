# “We” Sentences — 60s p5.js Animation

This sketch extracts and compares keywords from the provided Georgia Tech strategic plan PDF (via the text snapshot in `georgia-tech-strategic-plan-2020-2030.txt`) and plays an animated typography loop.

## Run the animation

Because the sketch loads `sentences.json`, you should run a local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in a browser.

## Re-extract the buzzword pairs + counts

`buzzword_pairs.json` is generated by counting how often each requested term/phrase appears.

```bash
python3 extract_buzzword_pairs.py
```

## Files

- `index.html`: page loader
- `sketch.js`: p5.js animation (60-second loop)
- `buzzword_pairs.json`: keyword pairs + counts (shown back-to-back)
- `extract_buzzword_pairs.py`: extraction script
- `georgia-tech-strategic-plan-2020-2030.txt`: text snapshot used for extraction
